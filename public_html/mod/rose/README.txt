CHOICE OF GRAPH LIBRARY


For now, we have selected these candidates:

pChart 2
JpGraph
RGraph

(3d.js is not included because it does not support MSIE8)

The server side libraries are slow, but reliable.
Interestingly, another dev company went through the same selection procedure:
http://www.winchesterinnovation.co.uk/2013/04/custom-reporting/

- and came up with the same shortlist, eventually selecting RGraph. I have noticed some glitches
in the radar chart for Google Chrome, in that library. We will take this library for a test drive though.
